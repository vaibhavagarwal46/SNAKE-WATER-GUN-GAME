#  Snake–Water–Gun Game (Python)

A simple **Snake–Water–Gun** command-line game built using **Python**. The player competes against the computer, which makes a random choice each round. The game then decides the winner based on predefined rules.

##  Game Rules

- **Snake drinks Water → Snake wins**
- **Water disables Gun → Water wins**
- **Gun kills Snake → Gun wins**
- Same choices result in a **draw**

##  Technologies Used

- **Python 3**
- **random module**

##  How to Run the Game

1. Make sure Python is installed
2. python snake_water_gun.py
